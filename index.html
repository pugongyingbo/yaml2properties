Properties 格式示例:

# 基础配置
server.port=8080
spring.application.name=demo

# 嵌套属性
spring.datasource.url=jdbc:mysql://localhost:3306/db
spring.datasource.username=root

# 数组示例
list[1]=item1
list[2]=item2

# 对象数组
users[1].name=张三
users[1].age=18
users[2].name=李四
users[2].age=20

YAML 格式示例:

# 基础配置
server:
  port: 8080
spring:
  application:
    name: demo
  datasource:
    url: jdbc:mysql://localhost:3306/db
    username: root

# 数组示例
list:
  - item1
  - item2

# 对象数组
users:
  - name: 张三
    age: 18
  - name: 李四
    age: 20
